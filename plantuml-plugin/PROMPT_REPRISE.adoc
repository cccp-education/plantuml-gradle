== Mission session 162 — Cross-provider fallback

Dernière tâche EPIC 13 (LOW) : fallback inter-provider quand un provider
est entièrement saturé. Exemple : Gemini Pro épuisé → Mistral Free → Ollama.
Actuellement `ApiKeyPool` est construit par provider (un pool OpenAI, un pool
Gemini, etc.) sans pont inter-provider. Objectif : orchestrer la rotation
cross-provider quand un pool entier est saturé.
