== EP-3 — Pool freemium ratio dynamique

=== Fait
- `FreemiumWeightCalculator.kt` créé (domaine DDD) — calcule poids dynamique basé sur quota restant + ratio freemium ✅
  - `freemiumRatio ∈ [0.0, 1.0]` : 0.0 = désactivé, 1.0 = 100% FREE
  - FREE keys : `weight × remainingRatio × freemiumRatio`
  - ENTERPRISE/PRO keys : `weight × remainingRatio × (1 - freemiumRatio)`
